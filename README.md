# Python Beginner Projects

This repository contains my beginner-level Python projects created while learning and practicing Python fundamentals.

The purpose of this repository is to document my learning progress, improve my programming logic, and build small practical projects step by step.

## Projects

The most notable projects in this repository are:

### 1. Number Guessing Game

An interactive command-line Python game where the user selects a number between 1 and 100, and the program tries to guess it.

The user provides feedback after each guess, indicating whether the selected number is higher or lower than the program's guess. The game continues until the program guesses the correct number or reaches the allowed number of attempts.

**Concepts used:**

* Loops
* Conditional statements
* User input
* Basic logic building
* Command-line interaction

### 2. To-Do List using File Handling

A command-line To-Do List application that allows the user to manage tasks. The tasks are stored in a `.json` file, so the data can be saved and reused later.

**Concepts used:**

* File handling
* JSON file storage
* Functions
* Lists
* User input
* Basic application structure

## How to Run

```bash

```

2. Open the project folder in VS Code, Jupyter Notebook, or any Python-supported IDE.

